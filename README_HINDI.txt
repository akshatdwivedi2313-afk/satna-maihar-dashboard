Satna CFT Dashboard

इस version में Cluster के बाद उपयंत्री filter जोड़ा गया है।
उपयंत्री values Excel के GP Main sheet से ली गई हैं।

चलाने का तरीका:
1. ZIP को पूरा extract करें।
2. पुराने dashboard folder की जगह नया extracted folder use करें।
3. start_dashboard.bat पर double click करें।
4. Browser में http://localhost:8080/index.html खुलेगा।
